a-stock-data

A股研报批量采集 · 东财 reportapi 驱动

GitHub 开源 Python CLI 399 篇实战采集 16 只标的覆盖

调用东方财富 reportapi（reportapi.eastmoney.com），按股票代码批量拉取券商深度研报。

核心参数 code= + qType=0 + pageSize=100

⚠️ 禁用关键词搜索 — q= 参数噪声极大（周报/不相关行业混入），按股票代码逐只拉取才能命中深度研报。

Python — 批量拉取示例
import requests, json, time, random

STOCKS = {
    "光模块": ["300308", "300502", "300394"],
    "GPU芯片": ["688041", "688256"],
    "PCB": ["002463", "002916", "300476"],
}

for sector, codes in STOCKS.items():
    for code in codes:
        params = {
            "industryCode": "*", "pageSize": "100",
            "beginTime": "2025-01-01", "endTime": "2026-12-31",
            "pageNo": "1", "qType": "0", "code": code,
        }
        r = requests.get(
            "https://reportapi.eastmoney.com/report/list",
            params=params,
            headers={"Referer": "https://data.eastmoney.com/"}
        )
        rows = r.json().get("data", [])
        print(f"{code} {sector}: {len(rows)} reports")
        time.sleep(1.0 + random.random() * 0.5)  # 节流

秘塔 Metaso

AI 搜索引擎 · MCP 协议接入

3 工具 search/reader/chat MCP 标准协议 6路并行 新闻雷达

通过 MCP 协议提供 web_search（搜索）、web_reader（全文阅读）、chat（RAG 问答）三个工具。

⚠️ 参数注意 search 用 q=，chat 用 message=，不要混用 query=。

角色 国内信源的主要入口 — 36氪、投资界等 SPA 站点无法从美国 IP 直连，走秘塔搜索获取内容。

Python — MCP 调用示例
import json, httpx

url = "https://metaso.cn/api/mcp"
headers = {"Authorization": "Bearer ",
           "Content-Type": "application/json"}

# 搜索
payload = {"jsonrpc": "2.0", "method": "tools/call",
           "params": {"name": "metaso_web_search",
                      "arguments": {"q": "AI算力 光模块 最新",
                                    "size": 10, "includeSummary": True}}}
r = httpx.post(url, json=payload, headers=headers)
result = json.loads(r.json()["result"]["content"][0]["text"])
